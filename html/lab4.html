  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    5. let, var, cont son locales dentro de una función. var no es local dentro de un bloque. Si un variable no esta declarado dentro de una función y simplemente se asignan tiene alcance global. Modo estricto de JavaScript previene ese comportamiento.
