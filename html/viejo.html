Lista de Preguntas y Respuestas

¿Cuál es la diferencia entre Internet y la World Wide Web?

La internet existió antes de la WWW. Es una red de computadoras que están interconectadas. Es una infraestructura para poder conectar todo junto. La WWW es algo que existe por encima de la internet. Es un sistema de documentos, páginas web, que se puede acceder a través de un protocolo como HTTP o HTTPS.

¿Cuáles son las partes de un URL?

Por ejemplo, una URL como https://experiencia21.tec.mx/ El HTTPS indica el protocolo, el "experiencia21" indica el subdominio, el "tec.mx" es el dominio. También existe la ruta que te lleva directo a un recurso dentro del servidor. La puerta, como ":8080" indica la entrada del servidor web.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Son métodos de petición para realizar unas acciones dentro de un recurso. GET - Solicita un recurso y solo debe recuperar datos. HEAD - Igual que un GET pero sin el cuerpo de la respuesta. POST - Envía una entidad a un recurso. Causa un cambio en el servidor. PUT - Reemplaza las representaciones al momento en un recurso. PATCH - Aplica modificaciones parciales a un recurso. DELETE - Borrar un recurso.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Se usaría un POST para mandar los detalles del usuario y contraseña. Esto permite que los datos se manden al servidor y se pueden procesar.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Se usa un GET para obtener los contenidos de la página. Esto solicita los recursos de la página y permite recuperar los datos.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Significa que la solicitud fue exitosa. Indica "OK" no hubo ningún error y se entregó la información esperada.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Puede ser error del desarrollador si fue un enlace roto dentro del sitio. Pero si fue escribir mal el URL entonces es falta del usuario.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Definitivamente, esto indica que fue un "Internal Server Error" así que es un fallo de servidor interno y se debe corregir.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Ya no es un estándar solito e incluye CSS. Un cambio clave son las etiquetas semánticas. Tiene mejor soporte multimedia para audio y video, incluso vectores SVG.

¿Qué componentes de estructura y estilo tiene una tabla?

Una tabla tiene varios componentes. Empieza con las etiquetas para empezar la tabla con <table>, también tiene <tr> para las filas, <th> para encabezados, <td> para celdas regulares. Los estilos de CSS se utilizan para cambiar el tamaño, padding, colores, bordes de las celdas. Se puede hacer responsive para que se adapte a la pantalla del usuario.
